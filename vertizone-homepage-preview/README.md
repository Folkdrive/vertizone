# Vertizone Homepage Preview

## Project Structure

- `index.html` — Main file with toggle buttons
- `style.css` — Main stylesheet
- `script.js` — Toggle functionality
- `assets/`
  - `logo/` — Logo and favicon
  - `images/` — All downloaded images go here
  - `videos/` — For animated backgrounds
  - `icons/` — For tech stack icons

## Instructions

1. Place your logo SVG and favicon in `assets/logo/`.
2. Add images to `assets/images/`.
3. Add videos to `assets/videos/`.
4. Add tech stack icons to `assets/icons/`.
5. Open `index.html` in your browser to preview the homepage.
